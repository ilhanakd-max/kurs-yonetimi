# Kurs Yönetimi Uygulaması

Bu depo, Çeşme Belediyesi Kültür Müdürlüğü için hazırlanan tek dosyalık PHP kurs takip uygulamasını içerir.

## Başlangıç
1. `schema.sql` dosyasını MySQL sunucunuzda çalıştırarak veritabanını oluşturun.
2. `index.php` dosyasını PHP 8+ destekleyen bir sunucuya yerleştirin.
3. Ortam değişkenleriyle bağlantıyı yapılandırın (opsiyonel):
   - `MYSQL_HOST` (varsayılan: `127.0.0.1`)
   - `MYSQL_DATABASE` (varsayılan: `kurs_yonetimi`)
   - `MYSQL_USER` (varsayılan: `root`)
   - `MYSQL_PASSWORD` (varsayılan: boş)

Varsayılan yönetici hesabı: `admin@example.com` / `admin123`.
